Chrome / Edge / Brave

Firefox
  1. Open Firefox and go to Settings (or type about:preferences)
  2. Search for "certificates" and click View Certificates
  3. Go to the Authorities tab
  4. Click Import...
  5. Select the downloaded cacert.pem file
  6. Check Trust this CA to identify websites
  7. Click OK
üçé macOS
üîë
System-wide (Safari, Chrome, Edge, Brave)
  1. Download the cacert.pem file
  2. Double-click the file to open Keychain Access
  3. Select System keychain if prompted
  4. Enter your admin password when prompted
  5. Find the certificate in Keychain Access and double-click it
  6. Expand Trust section
  7. Set When using this certificate to Always Trust
  8. Close the window and enter your password to save changes

üêß Linux
üíª
System-wide (Chrome, Edge, Brave)
  1. Download the certificate: wget /tls/cacert.pem
  2. Copy to system certificates: sudo cp cacert.pem /usr/local/share/ca-certificates/cacert.crt
  3. Update certificates: sudo update-ca-certificates
  4. Restart your browser
Note: Commands may vary by distribution. For RHEL/CentOS, use /etc/pki/ca-trust/source/anchors/ and update-ca-trust.

üì± iOS
üì≤
Safari (System-wide)
  1. Download the certificate using Safari
  2. When prompted, tap Allow to download the profile
  3. Go to Settings > General > VPN & Device Management
  4. Under Downloaded Profile, tap the certificate
  5. Tap Install and enter your passcode
  6. Tap Install again to confirm
  7. Go to Settings > General > About > Certificate Trust Settings
  8. Enable full trust for the certificate
ü§ñ Android
üì±
System-wide (Chrome, Edge, Brave, etc.)
  1. Download the cacert.pem file
  2. Go to Settings > Security (or Biometrics and security)
  3. Tap Other security settings > Install from device storage
  4. Navigate to and select the downloaded certificate file
  5. Enter a name for the certificate
  6. Select VPN and apps or WiFi usage
  7. Tap OK
Note: Menu names vary by Android version and manufacturer. Look for "Certificate" or "Credential" settings.
